# 📁 assets/menu/

Esta pasta contém as **imagens do Cardápio**.

## Como usar

Cada aba do cardápio (Drinks, Petiscos, Especiais, Sem Álcool) exibe uma
imagem decorativa ao lado da lista de itens.

## Imagens recomendadas

| Arquivo | Seção | Tamanho sugerido |
|---------|-------|-----------------|
| `drinks-flat.png` | Drinks & Sem Álcool | 800×600px |
| `food.png` | Petiscos | 800×600px |
| `bar.png` | Especiais da Casa | 800×600px |

## Como atualizar os itens do cardápio

1. Abra o `index.html`
2. Encontre o painel correspondente (ex: `id="panel-drinks"`)
3. Edite os `<div class="menu-item">` com nome, descrição e preço

## Formato

- **Formato**: JPG ou PNG
- **Tamanho**: 800px de largura mínima
- **Proporção**: 4:3 ou 16:9 (paisagem)
- **Dica**: Fotos com iluminação quente e fundo escuro combinam com a paleta da Môi

> Esta pasta é **separada** das outras para facilitar a atualização dos itens
> do cardápio sem afetar a galeria ou o hero.
