# 📁 assets/events/

Esta pasta contém as **imagens dos cards de Eventos**.

## Como usar

Cada evento tem um card com imagem de fundo. A imagem ocupa 100% do card,
com overlay escuro e texto sobreposto.

## Imagens recomendadas

| Arquivo | Evento | Tamanho sugerido |
|---------|--------|-----------------|
| `show1.png` | Môii Nights | 800×1100px |
| `show2.png` | Golden Hour | 800×1100px |
| `show3.png` | Noite Íntima | 800×1100px |

## Como adicionar novos eventos

1. Coloque a nova imagem aqui (ex: `show4.png`)
2. Edite o `index.html` — encontre a seção `#eventos`
3. Copie um `<article class="evento-card">` existente e ajuste os dados

## Formato

- **Formato**: JPG ou PNG
- **Tamanho**: 800px de largura mínima
- **Proporção**: 3:4 (vertical)
- **Qualidade**: Alta

> Dica: Imagens com pessoas, luzes e palco funcionam muito bem.
