# 📁 assets/hero/

Esta pasta contém as **imagens flutuantes do Hero** (seção principal da página).

## Como usar

Coloque suas imagens JPG/PNG aqui. Elas aparecerão flutuando no fundo do Hero
com efeito de parallax ao mover o mouse.

## Imagens recomendadas

| Arquivo | Descrição | Tamanho sugerido |
|---------|-----------|-----------------|
| `club.png` | Interior da boate | 800×1000px |
| `dj.png` | DJ ou show ao vivo | 800×1000px |
| `crowd.png` | Pista de dança / galera | 800×1000px |
| `drink.png` | Drink / coquetéis | 700×900px |

## Formato

- **Formato**: JPG ou PNG
- **Qualidade**: Alta (mínimo 72dpi)
- **Proporção ideal**: 3:4 ou 4:5 (vertical funciona melhor)

> Substitua os arquivos existentes (placeholders gerados com IA) pelos JPEGs reais
> da boate. Mantenha os mesmos nomes de arquivo para não precisar editar o HTML.
