# 📁 assets/logo/

Esta pasta contém o **logo oficial da Môi**.

## Arquivo principal

| Arquivo | Uso | Formato |
|---------|-----|---------|
| `logo-moi.png` | Navbar + Hero + Footer | PNG com fundo transparente |

## Dicas

- Use sempre o **PNG com fundo transparente** para que o logo funcione
  em qualquer fundo escuro
- O logo deve ter pelo menos **600px de largura** para garantir nitidez
  em telas Retina / HiDPI

## Onde o logo é usado

1. **Navbar** — canto superior esquerdo (altura: 44px)
2. **Hero** — centro da tela, exibido grande (até 460px de largura)
3. **Footer** — canto inferior esquerdo (altura: 36px)

> Para trocar o logo: substitua o arquivo `logo-moi.png` por sua versão atualizada.
> Não precisa alterar o HTML.
