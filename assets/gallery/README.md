# 📁 assets/gallery/

Esta pasta contém as **imagens da Galeria** exibidas na seção "Veja nosso espaço".

## Como usar

As imagens são exibidas em um mosaico clicável com lightbox (ampliação ao clicar).

## Imagens recomendadas

| Arquivo | Descrição | Tamanho sugerido |
|---------|-----------|-----------------|
| `club.png` | Vista geral da boate | 1200×800px |
| `dj.png` | Palco / DJ | 800×800px |
| `bar.png` | Bar / bancada | 1200×800px |
| `crowd.png` | Pista de dança | 800×800px |
| `lounge.png` | Área VIP / lounge | 800×800px |
| `drink.png` | Drinks / coquetéis | 800×800px |

## Formato

- **Formato**: JPG ou PNG
- **Qualidade**: Alta (mínimo 72dpi, preferível 150dpi)
- **Orientação**: Paisagem ou quadrado funciona melhor

> Substitua os arquivos existentes pelos JPEGs reais da Môi.
